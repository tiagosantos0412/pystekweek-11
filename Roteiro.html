Devemos criar este campo {% csrf_token %} para que o django crie um hash que protegerá nosso formulário contra falsificações Na views na função cadastrar vamos inserir duas verificações uma para que se a requisição for GET, ele renderiza a pagina de Cadastro E se a requisição for POST, vamos capturar e printar estas variáveis. def cadastrar(request): if request.method == "GET": return render(request, 'cadastrar.html') elif request.method == "POST": nome = request.POST.get('nome') cpf = request.POST.get('cpf') telefone = request.POST.get('telefone') email = request.POST.get('email') senha = request.POST.get('senha') print(nome, cpf, telefone, email, senha) return HttpResponse('Usário cadastrado com sucesso') Agora vamos cadastrar as informaçções capturadas das variáveis no banco de dados, através das models: No arquivo models.py crie uma classe com o nome Usuarios: class Usuarios(models.Model): Para que esta classe seja considerada uma representação de uma tabela em um banco de dados ela precisa herdar de models.Model Esta classe terá os seguintes atributos que serão as colunas da tabela: nome, cpf, telefone, email, senha, cliente.
